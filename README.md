# LegacyScreen
Arduino Sketch to Operate the LegacyScreen

## Usage

1. Upload sketch to Arduino
2. Connect servo on pin 9
3. send via Serial (9600 baud) 3byte values from 000 - 180
4. See the clock turn - win!
